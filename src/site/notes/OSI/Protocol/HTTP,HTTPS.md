الهدف :- طلب صفحات الويب و ايصالها اليك
HTTP 
 - Hypertext transfer protocol
 - layer 7
 - TCP
 - 80,8080
 - Markup language (HTML , CSS , XML)
يحتوي علي 3 انواع message :-
1- GET
 - طلب صفحات الويب
 2 - POST 
  - تستخدم لارسال بيانات مثل (username , password)
  3 - PUT
   - يتم استخدامها من قبل الادمن للتعديل في محتوي الصفحه

HTTPS
 - Hypertext transfer protocol secure
 - layer 7
 - TCP
 - 443
 - GET
 - POST
 - PUT
 - يدعم ال encryption
